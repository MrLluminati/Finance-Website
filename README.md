# Finance Website

A long-term finance intelligence platform inspired by the scale and utility of portals like Moneycontrol, but built with a trust-first, legally safer, learning-friendly, and SaaS-ready approach.

## Vision

To build a credible financial intelligence platform that helps users understand markets, companies, personal finance, investing concepts, business news, and financial regulations with clarity, transparency, and source-based information.

The long-term ambition is to evolve this into a trusted global finance platform. The short-term objective is to build a strong full-stack portfolio project before 2028 while keeping the initial budget as close to zero as possible.

## Product Philosophy

This project will not begin as a real-time stock-data terminal. Instead, it will start as a clean, reliable financial education and analysis platform and gradually evolve into a SaaS-style product with user accounts, watchlists, portfolio tools, calculators, structured company pages, and eventually licensed data integrations.

## Current Stage

## Local Development

Clone the repository:

```bash
git clone https://github.com/MrLluminati/Finance-Website.git
cd Finance-Website
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open:

```bash
http://localhost:3000
```

## Initial Constraints

- Initial budget: ₹0 or as close to ₹0 as possible.
- Primary goal: learning + portfolio + long-term product foundation.
- Avoid paid market-data dependency at the beginning.
- Avoid scraping NSE/BSE or copying protected content.
- Avoid investment recommendations, target prices, or buy/sell/hold calls unless proper regulatory compliance is obtained.
- Build in public-quality structure suitable for future job applications and investor/product discussions.

## Recommended Initial Tech Direction

- Frontend: Next.js
- Styling: Tailwind CSS
- Deployment: Vercel or similar free-tier hosting
- Database/Auth later: Supabase free tier or equivalent
- Content: Markdown/MDX initially
- Charts: Recharts or Chart.js
- Version control: GitHub

## First MVP Direction

Version 1 should include:

- homepage;
- finance news/editorial layout;
- beginner learning section;
- stock/company directory using manually curated data;
- basic company profile pages;
- SIP calculator;
- CAGR calculator;
- EMI calculator;
- disclaimer, terms, privacy policy;
- responsive mobile-first UI.

## Long-Term Direction

The intended evolution is:

1. Finance content website
2. Financial tools website
3. User-account based dashboard
4. Portfolio/watchlist SaaS
5. Data intelligence platform
6. Licensed market-data platform
7. Global trusted financial intelligence brand

## Important Disclaimer

This project is for education, information, and software development. It should not provide investment advice, securities recommendations, or regulated research without appropriate legal and regulatory review.
